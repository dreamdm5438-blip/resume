## ขั้นตอนการสร้าง Infographic ประวัติส่วนตัว

1. **รวบรวมข้อมูล** — ถามผู้ใช้เพื่อเก็บข้อมูลดังนี้:
   - ชื่อ-นามสกุล
   - อาชีพ / ตำแหน่ง
   - คติประจำใจ (Tagline)
   - การศึกษา (สถาบัน, สาขา, ปีที่จบ)
   - ประสบการณ์ทำงาน (บริษัท, ตำแหน่ง, ช่วงเวลา)
   - ทักษะ (Skills) พร้อมระดับความสามารถ (%)
   - ช่องทางติดต่อ (อีเมล, โทรศัพท์, LinkedIn, GitHub ฯลฯ)
   - สีที่ชอบ / theme ที่ต้องการ

2. **สร้างไฟล์ HTML Infographic** — สร้างไฟล์ `profile_infographic.html` ใน `c:\resume\` โดยมีองค์ประกอบ:
   - Header: รูปโปรไฟล์ (placeholder), ชื่อ, ตำแหน่ง, คติประจำใจ
   - Section การศึกษา: Timeline style
   - Section ประสบการณ์: Timeline style
   - Section ทักษะ: Progress bar / Skill badges
   - Section ติดต่อ: ไอคอน + ข้อมูลติดต่อ
   - ใช้ CSS animations, Google Fonts (Thai-friendly เช่น Sarabun หรือ Prompt)
   - Responsive design, สีสันสวยงามแบบ modern/premium

3. **เปิดดูผลลัพธ์** — เปิดไฟล์ใน browser เพื่อตรวจสอบ

4. **ปรับแต่งตามความต้องการ** — แก้ไขสี, layout, หรือเนื้อหาตามคำขอของผู้ใช้
